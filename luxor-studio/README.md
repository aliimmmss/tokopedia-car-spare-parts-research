# Luxor Studio

Product Management Dashboard for Commercial Vehicle Spare Parts

## Features

- **Photo Studio**: Remove backgrounds from product photos
- **Product Writer**: Generate AI-powered product descriptions
- **Product List**: Manage product database with CSV storage

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 3.4
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is in `dist/` directory.

## Design System

Nothing Design - Dark, minimal, precise:
- OLED black backgrounds
- Sharp corners (zero border-radius)
- Space Grotesk, Doto, Space Mono fonts
- Red accent (#FF3B30)
- Green status (#00D26A)
