## Competitor Tracker Template

### How to Use

When you get the product list, I'll search Tokopedia/Shopee and populate this:

| Part Number | Dealer Cost | Tkp Seller 1 | Tkp Price 1 | Tkp Seller 2 | Tkp Price 2 | Shopee Low | Est. Market | Your Margin % | Rank |
|-------------|-------------|--------------|-------------|--------------|-------------|------------|-------------|---------------|------|
| | | | | | | | | | |

### Legend:
- **Rank**: 1-3 (Top 3 sellers on Tokopedia)
- **Margin %**: `((Market_Avg - Dealer_Cost) / Dealer_Cost) * 100`
- **Action**: Undercut / Match / Premium / Avoid
