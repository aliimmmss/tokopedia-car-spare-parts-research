# 🏁 Implementation Action Checklist

Use this checklist as your roadmap from "Zero" to "First Sale."

## Phase 1: Legal & Compliance (Week 1)
- [ ] **Account Setup:** Create Tokopedia Seller account and verify KTP/Bank Account.
- [ ] **Product Audit:** List all products and cross-reference with "Prohibited Items" list.
- [ ] **Certification:**
    - [ ] Identify which products need SNI (e.g., Brake Pads).
    - [ ] Request SNI certificates from suppliers.
- [ ] **Documentation:** Create a digital folder for all supplier invoices (Proof of Authenticity).

## Phase 2: Technical Listing Setup (Week 2)
- [ ] **Photography:** 
    - [ ] Set up a white background (use a photo box or a clean wall).
    - [ ] Take 7 photos per product (Hero, Packaging, Logo, Part Number, Scale, Fitment, Table).
- [ ] **Copywriting:** 
    - [ ] Draft titles using the `[Brand] [Part] [Compatibility] - [Feature]` formula.
    - [ ] Write technical descriptions including a full compatibility list.
- [ ] **Attribute Mapping:** Fill in all "Product Specifications" in the seller dashboard.
- [ ] **Pricing:** Calculate costs + fees and set competitive "Market Entry" prices.

## Phase 3: Logistics & Fulfillment (Week 3)
- [ ] **Courier Configuration:** 
    - [ ] Enable JNE REG, J&T, and SiCepat.
    - [ ] **CRITICAL:** Enable JNE JTR and Cargo for heavy parts.
- [ ] **Packaging Procurement:**
    - [ ] Buy high-grade bubble wrap (min 2 layers).
    - [ ] Source various sizes of double-wall corrugated boxes.
    - [ ] Purchase "Fragile" and "Do Not Throw" stickers.
- [ ] **Insurance Strategy:** Set a rule to always insure items > Rp 500.000.

## Phase 4: Marketing & Growth (Week 4+)
- [ ] **TikTok Integration:** Link Tokopedia account to TikTok Shop Seller Center.
- [ ] **Catalog Sync:** Ensure all products are pushed to the TikTok Shop tab.
- [ ] **Content Calendar:**
    - [ ] Schedule 3 "Pain-Solution" videos per week.
    - [ ] Schedule 1 "Authenticity Test" video per week.
- [ ] **Affiliate Outreach:** 
    - [ ] Identify 5 automotive micro-influencers.
    - [ ] Send sample kits for review.
    - [ ] Set affiliate commission to 10-15%.
- [ ] **Live Stream:** Set a weekly "Compatibility Q&A" live session.
