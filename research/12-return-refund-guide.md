## 1. RETURN POLICY FOR AUTOMOTIVE PARTS

### Return Window Timeline

| Seller Type | Return Window | Notes |
|-------------|---------------|-------|
| **Non-Mall (Marketplace)** | **6 calendar days** after delivery | Standard for most sellers |
| **Mall Sellers** | **15 calendar days** after delivery | Higher buyer protection |
| **Selected Customers** | Up to 15 calendar days | Special program members |

### Valid Return Reasons for Automotive Parts

A customer may return a product if:

| Reason Category | Examples |
|-----------------|----------|
| **Product doesn't match description** | Wrong part number, different brand, specs differ from listing |
| **Wrong product sent** | Ordered brake pads for Avanza, received for Xenia |
| **Product defective/doesn't work** | Defective injector, broken filter, non-functional sensor |
| **Incomplete/missing accessories** | Missing gaskets, bolts, installation manual |
| **Product damaged in transit** | Crushed box, bent parts due to poor packaging |

### When Return is NOT Valid

| Situation | Seller Action |
|-----------|---------------|
| Buyer ordered wrong part (wrong fitment) | Can reject if listing clearly stated compatibility |
| Buyer changed mind / "No longer needed" | Only eligible if product labeled "Change of Mind" eligible |
| Product used / installed / seal broken | Reject — product not in resellable condition |
| Damage caused by buyer (improper installation) | Reject with evidence of proper packaging |
| Complaint after return window passed | Direct buyer to contact seller directly |

## 2. "BARANG TIDAK SESUAI" (Item Not As Described) — DEFINITION

### What Qualifies as "Barang Tidak Sesuai"

For automotive spare parts, this includes:

| Category | Examples |
|----------|----------|
| **Wrong Part Number** | Listed as OEM 17801-0D020, received 17801-0D030 |
| **Different Brand** | Listed as BOSCH, received generic/no-brand |
| **Specs Different** | Listed as EURO 4 injector, received EURO 2 |
| **Wrong Application** | Listed for Hino Ranger, received for Isuzu NMR |
| **Different Material** | Listed as ceramic brake pads, received semi-metallic |
| **Counterfeit Product** | Claimed genuine DENSO, received fake |

### What Does NOT Qualify

| Situation | Explanation |
|-----------|-------------|
| Buyer ordered wrong model | Buyer's mistake, not seller's error |
| Buyer misread compatibility | If listing clearly stated vehicle compatibility |
| Minor cosmetic difference | If product functions correctly and matches specs |
| Packaging changed | If product inside is correct |

## 3. DISPUTE HANDLING PROCESS

### Step-by-Step Dispute Flow

```
BUYER FILES COMPLAIN → SELLER REVIEW (2 days) → APPROVE/REJECT
                                                        ↓
                                              IF REJECTED:
                                                        ↓
                                    BUYER UPLOADS MORE EVIDENCE
                                                        ↓
                                    SELLER RECEIVES RETURN (if approved)
                                                        ↓
                                    SELLER REVIEWS RETURN (2 working days)
                                                        ↓
                                    APPROVE REFUND / REJECT / ESCALATE
```

### Seller Response Timeline

| Stage | Timeline | Consequence of Missed Deadline |
|-------|----------|-------------------------------|
| **Initial Return Request** | 2 calendar days | **Auto-approved** |
| **Return Product Review** | 2 working days (pick-up) / 14 calendar days (self-arranged) | **Auto-approved** |
| **Appeal Submission** | 15 calendar days after refund | Cannot appeal |

### Platform Intervention

When buyer and seller cannot agree:

| Trigger | Process |
|---------|---------|
| Seller rejects request | Buyer can submit second request within 48 hours |
| Second rejection | Platform intervenes with dispute process |
| Platform review | Uses submitted materials + system data |
| Maximum disputes | **2 dispute requests per product** |

### Evidence Requirements for Disputes

| Claim Type | Evidence Seller Must Provide |
|------------|------------------------------|
| Product matches description | Packing video/photos showing correct item shipped |
| Product functions correctly | Product demonstration video + quality certificate |
| Product is authentic | Proof of authenticity (invoice, authorization letter) |
| Product was delivered | Delivery confirmation, tracking proof |
| Buyer used product | Photo evidence of installation marks, used condition |

## 5. REDUCING RETURNS — PREVENTING WRONG FITMENT

### Why Wrong Fitment Happens (Indonesian Context)

| Cause | Percentage | Prevention |
|-------|------------|------------|
| Buyer didn't check vehicle year/model | 40% | Require buyer to confirm in chat |
| Same car, different engine variant | 25% | List all compatible variants |
| Regional spec differences | 15% | Clarify EURO specs, local vs import |
| Vague product title | 10% | Include year, engine code, VIN |
| Platform miscategorization | 10% | Double-check category assignment |

### Prevention Strategies

#### A. Product Title Best Practices

**WRONG:**
> "Kampas Rem Toyota Avanza"

**CORRECT:**
> "Kampas Rem Depan Toyota Avanza 1.3 1.5 X E G 2004-2011 - Bendix"

#### B. Description Must Include

```
KOMPATIBILITAS:
✅ Toyota Avanza 1.3 E (2004-2011)
✅ Toyota Avanza 1.5 G (2004-2011)
✅ Toyota Avanza X (2004-2011)
❌ TIDAK COCOK untuk Avanza Xenia (Daihatsu)
❌ TIDAK COCOK untuk Avanza Veloz/Vanza 2012+

NOMOR PART:
OEM: 04465-28110 / Bendix BN-0455

CARA CEK PART YANG COCOK:
1. Cek tahun produksi mobil (STNK)
2. Cek kode mesin di mesin (K3-VE / 3SZ-VE)
3. Konfirmasi via chat sebelum beli
```

#### C. Pre-Purchase Verification

| Method | Implementation |
|--------|----------------|
| **Chat template** | "Sebelum order, mohon konfirmasi tahun mobil & kode mesin" |
| **Auto-reply** | "Untuk kecocokan part, silakan chat foto STNK/kerja mesin" |
| **Required field** | Use Tokopedia's custom note feature (if available) |
| **Product variants** | Create separate SKUs for each vehicle variant |

#### D. Warning Banners in Description

```
⚠️ PERHATIAN:
- Pastikan tahun mobil Anda sesuai sebelum order
- Part ini TIDAK cocok untuk model facelift tahun 2012+
- Jika ragu, CHAT kami dengan foto STNK untuk konfirmasi
- Return karena salah order BUKAN tanggung jawab seller
```

## 11. COMMON SCENARIOS & SOLUTIONS

### Scenario 1: Buyer Claims Wrong Part Number

**Buyer says:** "Yang dikirim part number beda, 17801-0D030 bukan 17801-0D020"

**Your Action:**
1. Check packing video — did you ship correct part?
2. If YES: Show packing video as proof, reject complaint
3. If NO: Accept return, apologize, offer replacement
4. If part numbers are interchangeable: Explain in chat with documentation

### Scenario 2: Buyer Claims Part Doesn't Fit

**Buyer says:** "Kampas rem tidak muat di motor saya"

**Your Action:**
1. Check listing — was compatibility clearly stated?
2. Request photo of vehicle/model
3. If buyer ordered wrong model: Reject with listing screenshot
4. If listing was vague: Consider accepting return to maintain rating

### Scenario 3: Buyer Claims Defective After Installation

**Buyer says:** "Rem bunyi setelah dipasang, produk rusak"

**Your Action:**
1. Request photo of installed condition
2. Ask: Was installation done correctly?
3. If product has installation marks: Can argue improper installation
4. Request mechanic's assessment
5. If genuinely defective: Accept return

### Scenario 4: Buyer Claims Counterfeit

**Buyer says:** "Ini bukan DENSO original, kualitasnya jelek"

**Your Action:**
1. Prepare proof of authenticity (supplier invoice, authorization)
2. Request specific complaint about quality
3. Provide documentation showing genuine product
4. If buyer persists: Let platform decide with your evidence

### Scenario 5: Buyer Returns Used Product

**Buyer says:** Returns product after using/installing it

**Your Action:**
1. Document unboxing of return (video)
2. Photograph installation marks, scratches, seal broken
3. Reject refund with evidence of used condition
4. Escalate to platform if buyer disputes
